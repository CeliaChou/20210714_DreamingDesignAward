徵稿格式

  1. 投稿作品需包含「玩家角色名稱」、「時裝設計理念說明」、「時裝名稱」。
  2. 圖檔格式限 JPG、PNG ( 300dpi以上 ) ( 總容量限制25MB內 )，可以傳送壓縮檔( 限ZIP或RAR格式 ) 或 google雲端連結。
  3. 時裝設計部位需包含「帽子/頭飾」、「服裝」、「手套」、「靴子」 4個部分，若有缺失的部分請於設計理念中說明不加入的原因，若未說明將視同不符合規定。

  5. 投稿作品需為彩色稿件，恕不接受黑白稿件。
  6. 設計範本含男性/女性 角色各一張，男女角色皆需設計。

徵稿格式

  1. 完成投稿的玩家，官方將於7天內透過客服信發送是否參加成功的信件通知。
  2. 若未參加成功，可於收到信件後3日內進行補件，若未於時間內進行補件視同放棄報名、參賽資格。
  3. 投稿作品與投稿所填寫之資料一經投遞送出後，如欲變更或修改，須於報名投稿時間截止前3日內寄送客服信件告知，逾期不受理。
  4. 參賽者須保證投稿作品之名稱、圖片、作品或內容嚴禁含有非法、政治性、侵害隱私、妨礙風化之內容或其他違反法規等情事。
  5. 投稿作品須為參賽者自行獨立創作、具有原創性，嚴禁抄襲、複製、仿冒或侵犯他人著作權。
  6. 參加者同意授權掘夢使用參加本活動所提供之作品、說明文字其他資料，做為活動相關宣傳使用。
